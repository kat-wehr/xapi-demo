Mapping Competencies to
Formative Microlearning

How to append program outcome and competency labels to xAPI statements generated by formative activities — enabling continuous, evidence-based performance tracking beyond summative assessments.

Core Concept
Input
Formative Microlearning Activity
Quiz, simulation, branching scenario, drag-and-drop, reflection prompt, etc.
Mechanism
xAPI Statement + Extensions
Actor · Verb · Object + context.extensions with competency/outcome IDs attached
Output
Competency Performance Record
Aggregated evidence toward program & course outcomes stored in your LRS

Implementation Workflow
01
Map Activities to Outcomes
Align each microlearning activity to one or more program outcomes, course outcomes, and/or competencies in a mapping document or LMS metadata.
›
02
Configure xAPI Extensions
Add context.extensions to each activity's xAPI template with your institution's outcome/competency IRIs or ID strings.
›
03
Learner Completes Activity
The authoring tool (Storyline, Rise, H5P, etc.) fires the xAPI statement with outcome labels embedded in every response.
›
04
Statements Land in LRS
Your Learning Record Store (e.g., SCORM Cloud, Watershed, Learning Locker) stores each statement with its competency metadata intact.
›
05
Query & Aggregate
Filter LRS data by outcome/competency ID across all formative touchpoints to build a continuous performance picture per learner.
What You Can Now Measure
📈
Competency Trajectory
Track how a learner's performance on a specific competency evolves across multiple formative activities over time — not just the final exam score.
🔍
Early Warning Signals
Identify learners struggling with a program outcome before summative assessment — triggering timely intervention and support.
🗺️
Coverage Gaps
Spot which outcomes have few or no formative touchpoints, revealing blind spots in your instructional design.
🏛️
Accreditation Evidence
Produce granular, timestamped evidence of learner performance toward program outcomes — directly supporting accreditation reporting.
⚖️
Formative vs Summative Comparison
Compare formative activity scores against final assessments per competency to evaluate predictive validity of your microlearning design.
🎯
Cohort Benchmarking
Aggregate across learners to see which competencies a cohort masters quickly vs. where instructional redesign is needed most.
